# Weather App 🌤️

A simple weather application that allows users to check current weather conditions by searching for a city.

## 🚀 Features
- Search weather by city name
- Display temperature, weather condition, and location
- Real-time API data fetching
- Clean and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

- ## 🔗 API Used
OpenWeather API

## 📌 Project Goal
This project was built to practice working with APIs, asynchronous JavaScript (fetch), and handling real-time data.
